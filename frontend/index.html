Architecture

5-Stage OCR Pipeline

From raw scanned image to clean Unicode text — OpenCV preprocessing meets Tesseract LSTM engine

📥
1
Image Ingestion
Load JPG/PNG/BMP/TIFF/WEBP. Auto color-space conversion to BGR.
🔬
2
Preprocessing
Upscale → CLAHE → FastNLM Denoise → Deskew → Adaptive Binarize → Morph cleanup.
🗺️
3
Script Detection
Identify Brahmic, Perso-Arabic, Dravidian or Latin script family.
🧠
4
Tesseract OCR
OEM 3 LSTM engine. PSM-configurable. Full image + line-level region OCR.
✅
5
Post-Processing
Unicode NFC normalize, RTL direction, noise artifact removal.

Quick Start

Start the API Server

3 steps — works on Windows, macOS, Linux with Python 3.8 to 3.14

1
Install Python Packages
No TensorFlow needed. Works on Python 3.14:
pip install -r backend/requirements.txt
Or install manually:
pip install fastapi uvicorn[standard] \ opencv-python pytesseract Pillow \ numpy loguru python-multipart \ python-dotenv pydantic aiofiles
2
Install Tesseract OCR
Windows: Download from UB-Mannheim, then create backend/.env:
TESSERACT_PATH=C:\Program Files\Tesseract-OCR\tesseract.exe
Ubuntu / Debian:
sudo apt install tesseract-ocr \ tesseract-ocr-hin tesseract-ocr-pan \ tesseract-ocr-urd tesseract-ocr-ben \ tesseract-ocr-guj tesseract-ocr-tam \ tesseract-ocr-tel tesseract-ocr-kan \ tesseract-ocr-mal tesseract-ocr-ori \ tesseract-ocr-mar tesseract-ocr-san
macOS:
brew install tesseract tesseract-lang
3
Run the Server
From the project root folder:
cd backend uvicorn app:app --reload --port 8000
Verify it's working:
# Health check (Tesseract status) http://localhost:8000/health # Interactive API docs (Swagger) http://localhost:8000/docs # Serve this web UI http://localhost:8000/ui
✅ Green dot in navbar = server ready. Open frontend/index.html and upload your image!
